Que hace el programa?

 Define 4 funciones. El método Main pide datos al usuario y llama a cada función para calcular: la suma de 2 números, verificar si es mayor de edad, el área de un círculo y el área de un triángulo.

 Que datos recibe de entrada?
 
1. Dos números enteros para la suma.
2. Un número entero para la edad.
3. Un número decimal para el radio del círculo.
4. Dos números para la base y altura del triángulo.

Que datos procesa?

El programa procesa los datos usando las siguientes funciones:
- Suma: num1 + num2
- Verificación de edad: edad >= 18
- Área del círculo: π * radio * radio
- Área del triángulo: (base * altura) / 2

- Que datos imprime?
  
Al finalizar imprime en pantalla:
- El resultado de la suma de los dos números
- Un mensaje indicando si es mayor o menor de edad
- El área calculada del círculo
- El área calculada del triángulo
